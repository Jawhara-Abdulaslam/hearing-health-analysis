 Hearing Health Survey Analysis

This project analyzes responses from a hearing well-being survey to uncover:

- Key barriers preventing people from getting hearing tests
- Most desired features in hearing support apps
- Public interest in using and paying for hearing-related mobile apps

 Project Structure

- Hearing_Health_Analysis.ipynb: Main notebook with data analysis, visualization, and insights
- Download the original survey data (`CSV`) from kaggle
- requirements.txt: Required Python packages
- README.md : You are here!
